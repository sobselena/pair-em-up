## 🚀 Deploy 
[Deploy](https://sobselena-pair-em-up.netlify.app/)

### App Installation 1. **Clone the repository**
```bash
git clone https://github.com/sobselena/pair-em-up.git
```
2. **Install dependencies**
```bash
npm install
```
3. **Start the app**
```bash
npm run dev
```
